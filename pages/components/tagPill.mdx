# TagPill

## Usage

```tsx

export const TagPills = () => (
<TagPill primary>Primary</TagPill>
<TagPill secondary>Secondary</TagPill>
<TagPill disabled>Disabled</TagPill>
<TagPill excluded>Excluded</TagPill>
<TagPill positive>Positive</TagPill>
<TagPill neutral>Neutral</TagPill>
<TagPill negative>Negative</TagPill>
);
```
